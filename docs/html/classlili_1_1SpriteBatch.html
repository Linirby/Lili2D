Detailed Description

Batches multiple sprites into a single draw call.

SpriteBatch is optimized for rendering many tiles or sprites that share the same texture (AtlasMap). Call begin() to start building the batch, draw() to add sprites, and end() to update the GPU mesh. Finally, call render() to submit the batch to the renderer.
